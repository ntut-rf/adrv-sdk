# Setup and Build

## Prepare SISO2 repository

Clone repository 
```shell-session
$ git clone git@github.com:ntut-rf/SISO2.git
```

Update submodules
```shell-session
$ cd SISO2
$ git submodule update --init
```

<!-- slide:break -->

# 

## Build

```shell-session
$ make -j
```

Wait for the build to complete.
